# Personal Portfolio Website

A modern and responsive personal portfolio website built using React, Vite, Tailwind CSS, and JavaScript. This portfolio showcases my skills, projects, education, and contact information while providing a clean and professional user experience.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Dark and Light mode support
* Animated Hero section
* About Me section
* Skills showcase
* Projects section with GitHub links
* Testimonials section
* Contact form with EmailJS integration
* Resume download option
* Smooth scrolling navigation
* Modern UI with Tailwind CSS

## Technologies Used

* React.js
* Vite
* Tailwind CSS
* JavaScript (ES6+)
* Framer Motion
* React Icons
* EmailJS

## Installation

1. Clone the repository

```bash
git clone https://github.com/gayuelango15-web/Portfolio.git
```

2. Navigate to the project folder

```bash
cd Portfolio
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
